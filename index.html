Projects

TimeMate: API-first Time Tracker

I faced my personal nightmare—datetime—and built an entire API-first time tracking backend to finally make peace with it. This project was built from scratch, not a tutorial clone.

  • Architecture: Clean, modular, and built on SOLID principles.
  • Performance: Intelligent caching with Redis to speed up frequent queries.
  • Reliability: Over 100 unit and integration tests with 99% code coverage.
  • Security: UUIDs as primary keys, DB-level constraints, and token-based auth.
View code on GitHub

Redmine Onboarding Automation (Proof of Concept)

An application created on my own initiative in response to a real business problem identified while working at Papu.io.

  • Goal: To prove the possibility of automating a time-consuming and error-prone process.
  • Result: Successfully reduced the number of manual steps from ~80 to 2, shortening the estimated operation time from 30 to 5 minutes.
